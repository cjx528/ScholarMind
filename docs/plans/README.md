# 后续计划

历史计划文档已经清理，只保留当前版本还能继续推进的方向。

## 1. 多源推荐解释增强

- 在推荐结果中展示召回来源。
- 对推荐排序和跳过原因给出更稳定的证据。
- 引入更明确的融合和重排解释。
- 让用户反馈可以更直观地影响后续排序。

## 2. 主题归档增强

- 把主题归档进一步升级成 query profile。
- 每个主题记录关键词、意图 query、论文源选择和最近入库结果。
- 给抓取失败增加可读错误原因。

## 3. PDF 阅读助手增强

- 增加当前会话历史持久化。
- 支持最近问题和置顶问题。
- 解析报告追问历史可回看。
- 回答中显示使用了选中文本、粗读、精读、推理链还是 PDF 正文。

## 4. 报告和答辩增强

- 按 A/B/C/D 分工补充截图和测试结果。
- 为每个核心功能准备一个成功案例和一个已知限制。
- 保留真实演示数据，避免现场依赖临时抓取。
