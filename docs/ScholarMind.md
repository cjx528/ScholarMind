# ScholarMind 产品概览

## 一句话定位

ScholarMind 是一个面向科研人员和学生的论文助手，帮助用户每天发现值得读的论文，在阅读时持续提问，并把阅读结果沉淀成长期知识库。

## 用户流程

1. 配置研究画像。
2. 建立主题订阅和论文源。
3. 查看每日研究雷达。
4. 进入论文库和论文详情。
5. 生成粗读、精读和推理链。
6. 在 PDF 阅读器中划词翻译、解释、总结和追问。

## 核心模块

### 推荐发现

- 用户画像。
- 主题订阅。
- 多源论文抓取。
- 每日研究雷达。
- 推荐反馈。

### 阅读理解

- 论文库。
- 论文详情。
- PDF 阅读器。
- 论文 AI 助手。
- 粗读、精读、推理链。
- 相似论文。

### 知识沉淀

- 单篇论文追问。
- 全库问答。
- Wiki。
- Agent。

### 配置运维

- 全局 LLM 设置。
- 登录认证。
- 任务状态。
- 成本统计。

## 当前实现边界

- v1 不做聊天记录永久保存，PDF 助手会话主要保存在当前页面。
- v1 不做流式输出，优先保证普通请求响应稳定。
- v1 不维护旧双栏译文页，翻译能力统一放进 PDF AI 助手。
- v1 不展示旧视觉说明入口，避免给出抽象且不可验证的解读。

## 后续可改进方向

- 增强聊天持久化和解析追问历史。
- 让研究雷达显式展示召回、融合、重排和最终分区理由。
- 给论文源抓取增加更细的失败原因和重试提示。
- 为推荐反馈设计更直观的排序变化解释。
- 增加多人协作标注和共享阅读记录。
